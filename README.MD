## SET UP
1. install dependency 
```bash
npm install
```
2. set environment variable 
```env 
SQL_USERNAME='your_sql_user'
SQL_PASSWORD='password_of_user'
SQL_HOST='host_ip_of_server'
SQL_DATABASE='database_name'

PORT=port_no

JWT_SECRET= 'your_jwt_code'
```
3. start api
```bash

# for test 
npm test
# for development 
npm run dev
# for start api
npm start

```
